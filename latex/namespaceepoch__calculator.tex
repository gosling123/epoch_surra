Documentation for \doxyref{epoch\+\_\+calculator}{p.}{namespaceepoch__calculator} module

The epoch\+\_\+calaculator module reads the E\+P\+O\+CH data files and calculates several parameters. There are three classes, one which handles the calculations from grid quantities (\doxyref{Laser\+\_\+\+Plasma\+\_\+\+Params}{p.}{classepoch__calculator_1_1_laser___plasma___params}), field quantities (E\+M\+\_\+fileds) and the momentum distribution function (\doxyref{dist\+\_\+f}{p.}{classepoch__calculator_1_1dist__f}).
